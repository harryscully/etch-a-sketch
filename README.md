# Etch-a-Sketch ✏️🖼️

A browser-based drawing toy built with vanilla JavaScript.

**Live demo:** https://harryscully.github.io/etch-a-sketch/

## Features

- Draw by hovering over the grid
- Adjustable grid size up to 64x64
- Rainbow mode — each cell fills with a random colour
- Opacity mode — cells gradually darken with each pass
- Clear button to reset the canvas

## Built with

- HTML
- CSS
- JavaScript

## What I learned

This was a good exercise in dynamic DOM manipulation — generating grid cells programmatically, handling mouseover events, and managing multiple toggle states. The opacity mode in particular required thinking carefully about how state accumulates across interactions.
